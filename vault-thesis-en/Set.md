# Set

A collection of distinct objects, considered as a whole. A primitive concept of [[Set Theory]] and the foundation of all mathematics.

---

## Definition

A set is defined by its elements. Two sets are equal if and only if they contain the same elements. Order does not matter, and there is no repetition.

Notation: S = {a, b, c} or S = {x | P(x)} (the set of all x satisfying property P).

## In the Thesis

The [[Sample Space]] S is a set -- the set of all [[Vector|vectors]] of the [[Matrix M]]. [[Context]] defines a [[Subset]] A ⊂ S. [[Tool|Tools]] define additional subsets.

[[Tautology]] emerges when a property P holds for all elements of the relevant set -- P is universally true within A.

[[Hallucination]] occurs when the model operates in the complement S \ A -- outside the set defined by the context.
