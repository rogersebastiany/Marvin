# Set

A collection of distinct objects, considered as a whole. A primitive concept of [[Teoria dos Conjuntos]] and the foundation of all mathematics.

---

## Definition

A set is defined by its elements. Two sets are equal if and only if they contain the same elements. Order does not matter, and there is no repetition.

Notation: S = {a, b, c} or S = {x | P(x)} (the set of all x satisfying property P).

## In the Thesis

The [[Espaço Amostral]] S is a set -- the set of all [[Vetor|vectors]] of the [[Matriz M]]. [[Contexto]] defines a [[Subconjunto]] A ⊂ S. [[Tool|Tools]] define additional subsets.

[[Tautologia]] emerges when a property P holds for all elements of the relevant set -- P is universally true within A.

[[Alucinação]] occurs when the model operates in the complement S \ A -- outside the set defined by the context.
