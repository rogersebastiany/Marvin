# Grafo Dirigido Completo

Um grafo onde todo par de vértices tem arestas dirigidas. Cada nó é alcançável a partir de qualquer outro nó. É "totalmente descobrível" — tautologicamente.

---

## Definição

Um grafo dirigido completo (complete digraph) é um grafo onde entre cada par de vértices existe pelo menos uma aresta dirigida em cada sentido.

Se para todos os membros do conjunto "nós" a propriedade "tem aresta direcional para todo outro nó" vale, então essa propriedade é [[Tautologia|tautológica]] nesse domínio.

## Relação com a Tese

O conceito de grafo dirigido completo foi o ponto de partida da tese [[Tautologia Ontológica]]. A pergunta original: "se todos os nodos tiverem arestas direcionais para outro nodo, é um grafo direcionado totalmente descobrível, sabe? Por tautologia."

A [[Ontologia]] do grafo (definição de nós, arestas, direção) permite a conclusão [[Tautologia|tautológica]] de que o grafo é completo. A ontologia é a causa, a tautologia é o efeito.

## Strongly Connected

Um grafo fortemente conectado (strongly connected) é onde a partir de qualquer nó você alcança qualquer outro nó seguindo as arestas. É a propriedade de "totalmente descobrível" — e quando isso vale para todos os nós, é tautológico.

Na tese: um sistema com [[Ontologia]] completa é como um grafo fortemente conectado — todo conceito é alcançável a partir de qualquer outro, sem lacunas.
